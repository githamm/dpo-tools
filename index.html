Tabletop to DataTables

Use A mostly responsive sortable table that updates dynamically from a public-facing Google sheet, based on Tabletop.js and DataTables, a jQuery plugin / Live example

GitHub Code for tables

Documentation Tabletop to DataTables

Steps

  1. Create new Google sheet in public-facing account
  2. Make it public: Share > Advanced > Change (from Private) > On - Public on the web > Save > Done
  3. Publish it
  4. Create new instances of "example.html" and "tabletop-feed.js"
  5. Open "tabletop-feed.js" in a text editor and add the Google sheet's key to line 7:

    initializeTabletopObject('SPREADSHEETKEY');

  6. Starting on line 28, add in spreadsheet's column headers (mDataProp value)* and how you want them displayed on the page (sTitle value); add new lines as necessary to match spreadsheet columns (note comma after each line except the last): 

        var tableColumns =   [
        {'mDataProp': 'athlete', 'sTitle': 'Athlete', 'sClass': 'center'},
        {'mDataProp': 'country', 'sTitle': 'Country', 'sClass': 'center'},
        {'mDataProp': 'event', 'sTitle': 'Event', 'sClass': 'center'},
        {'mDataProp': 'medal', 'sTitle': 'Medal', 'sClass': 'center'}
    ];

    * Do not use spaces in this string. Ex: {'mDataProp': 'skievent', 'sTitle': 'Ski Event', 'sClass': 'center'},

  7. Upload "example.html" (rename this at will) and "tabletop-feed.js" to Extras server
