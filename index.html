Charts

Custom Google charts

USE Quick, interactive chart to embed in NGPS freeform

TYPES Variety

CODE Code for line, bar, column and pie charts on GitHub

NOTE To change easily between pie, bar and column charts, change the chart type in line 36 (PieChart, BarChart or ColumnChart):

var chart = new google.visualization.PieChart(document.getElementById('chart_div'))

DOCUMENTATION Google charts

STEPS

  1. Copy chart code from GitHub and create new file with it
  2. Replace label and data variables

    3. Line chart example:

    data.addColumn('string', 'County');
        data.addColumn('number', 'Wells');
        data.addRows([
          ['Weld', 500],
          ['Larimer', 400],
          ['Yuma', 300],
          ['Washington', 200],
          ['Archuleta', 100]
        ]);

EMBED To embed a Google chart in a freeform, copy all the code, delete these tags: <html> <head> </head> <body> </body> </html> and paste remaining code into the freeform

OTHERS Quartz ChartBuilder, infogr.am (live example)

Chart.js

USE Interactive, responsive charts that can be added to NGPS via iframe — code below / Live example

TYPES Bar / Line / Doughnut / Pie / Polar / Radar

CODE Code for all charts on GitHub

DOCUMENTATION chartjs.org

IFRAME CODE EXAMPLE

<iframe height="400" width="654" frameBorder="0" src="http://URLTOCHART"></iframe>

STEPS

  1. Copy chart code from GitHub and create new file with it
  2. Replace label and data variables

    3. Line chart example:

    // Enter labels for horizontal axis
        labels : ["January","February","March","April","May","June","July"],
        datasets : [
            {
                fillColor : "rgba(72,99,160,0.5)", /* bar color */
                strokeColor : "rgba(72,99,160,0.8)", /* bar border color */
                highlightFill: "rgba(72,99,160,0.75)", /* bar color on hover/click */
                highlightStroke: "rgba(72,99,160,1)", /* bar border color on hover/click */
                data : ["1","3","5","7","9","11","13"] /* data */
            },
            // copy and paste above code for more bars; last set has no comma
            {
                fillColor : "rgba(161,72,99,0.5)",
                strokeColor : "rgba(161,72,99,0.8)",
                highlightFill : "rgba(161,72,99,0.75)",
                highlightStroke : "rgba(161,72,99,1)",
                data : ["2","4","6","8","10","12","14"]
            },
            {
                fillColor : "rgba(99,161,72,0.5)",
                strokeColor : "rgba(99,161,72,0.8)",
                highlightFill : "rgba(99,161,72,0.75)",
                highlightStroke : "rgba(99,161,72,1)",
                data : ["3","5","7","9","11","13","15"]
            }
        ]
  3. Upload file to Extras server and add chart via iframe or add code to web page (can't add code to article or freeform in NGPS because chart uses <canvas> tag)

Tables

DataTables

USE A responsive, sortable, searchable table that updates dynamically from a public-facing Google sheet, based on DataTables / Live example

CODE GitHub

STEPS

  1. Create new Google sheet in public-facing account
    • Google account: denverpostdata (password in Passpack)
  2. Make it public: Share > Advanced > Change (from Private) > On - Public on the web > Save > Done
  3. Publish it: File > Publish to the web > Publish
  4. Create new instances of example.html (GitHub code) and example.js (from "js" folder - GitHub code) and name them something relevant to your project
  5. Open the renamed example.html in a text editor and change the table headers (starting on line 45) to how you want them to display on the page: 

    <th>Header 1</th>
    <th>Header 2</th>
    <th>Header 3</th>
    <th>Header 4</th>
    <th>Header 5</th>

  6. Starting on line 62, change the data-title to match the spreadsheet headers: 

    <td data-title="Header 1:">{{ header1 }}</td>
    <td data-title="Header 2:">{{ header2 }}</td>
    <td data-title="Header 3:">{{ header3 }}</td>
    <td data-title="Header 4:">{{ header4 }}</td>
  7. On line 72, change the file name to match the .js file name from step 4

    <script src="http://extras.denverpost.com/frameworks/tables/datatables/js/example.js"></script>

  8. Open renamed example.js in a text editor and add your Google sheet's key* between the single quote marks on line 7:

    var spreadsheet_key = '1T0n6-bVZo2lpvL95HejOX9eKwCyH5g6Hc3XX41Yk4Yo';

    * Sample spreadsheet URL with key in red:

    https://docs.google.com/spreadsheets/d/1T0n6-bVZo2lpvL95HejOX9eKwCyH5g6Hc3XX41Yk4Yo/edit#gid=0

  9. Starting on line 77, change width percentages and add/remove { ... } to match table (note no comma after last { ... }):

    {
        "sWidth": "20%"
        // "sType": "formatted-num" 
    },{
        "sWidth": "20%"
        // "sType": "formatted-num" 
    },{
        "sWidth": "20%"
        // "sType": "formatted-num" 
    },{
        "sWidth": "20%"
        // "sType": "formatted-num" 
    },{
        "sWidth": "20%"
        // "sType": "formatted-num" 
    }
  10. Upload both files to the Extras server: frameworks > tables > datatables (.html file) and frameworks > tables > datatables > js (.js file)
  11. Table should be at http://extras.denverpost.com/frameworks/tables/datatables/YOURFILENAME.html

Maps

CartoDB

USE Super flexible, responsive, maps. You can drop .csv files of data points or shapefiles right into interface for easy mapping. Features the ability to combine tables (say, a shapefile of Colorado counties with Census income data) and do SQL-type filtering. / Live example

TYPES Simple, cluster, choropleth, category, bubble, torque, torque heat, torque category, intensity, density

DOCUMENTATION CartoDB
